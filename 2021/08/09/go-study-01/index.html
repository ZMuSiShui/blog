环境安装

本机系统：Windows 11 专业版 21H2

Go 版本：1.16.5 windows/amd64

  • 下载安装包

    地址：https://golang.org/dl/go1.16.5.windows-amd64.msi

  • 安装完成后检查

  • 检查 Go env

  • Hello World

    命令行执行：

    Go 命令

  • go run main

    不生成任何文件只运行程序。

  • go build main

    在对应当前目录下编译生成文件。

  • go install main

    会把编译好的结果移动到 $GOPATH/bin。

  • go fmt main

    格式化代码，将代码修改成标准格式。

常量与变量

数据类型

  • 空值：nil

  • 整型：

    • int8, int16, int32, int64
    • uint8, uint16, uint32, uint64
    • int, uint, 具体长度取决于 CPU 位数

  • 浮点数：

    • float32
    • float64

  • 字节：byte (等价于uint8)

  • 字符串：string

    只能用一对双引号（””）或反引号（``）括起来定义，不能用单引号（’’）定义！

  • 布尔值：boolean

    只有 true 和 false，默认为 false。

常量声明

常量，在程序编译阶段就确定下来的值，而程序在运行时无法改变该值。

单个常量声明

  • const 变量名称 数据类型 = 变量值

    如果不赋值，使用的是该数据类型的默认值。

  • const 变量名称 = 变量值

    根据变量值，自行判断数据类型。

多个常量声明

  • const 变量名称,变量名称 … ,数据类型 = 变量值,变量值 …
  • const 变量名称,变量名称 … = 变量值,变量值 …

变量声明

单个变量声明

  • var 变量名称 数据类型 = 变量值

    如果不赋值，使用的是该数据类型的默认值。

  • var 变量名称 = 变量值

    根据变量值，自行判断数据类型。

  • 变量名称 := 变量值

    省略了 var 和数据类型，变量名称一定要是未声明过的。

多个变量声明

  • var 变量名称,变量名称 … ,数据类型 = 变量值,变量值 …
  • var 变量名称,变量名称 … = 变量值,变量值 …
  • 变量名称,变量名称 … := 变量值,变量值 …

输出方法

  • fmt.Print

    输出到控制台（不换行）

  • fmt.Println

    输出到控制台并换行

  • fmt.Printf

    仅输出格式化的字符串和字符串变量（整型和整型变量不可以）

  • fmt.Sprintf

    格式化并返回一个字符串，不输出
